## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
   git clone https://github.com/yourusername/smartshift.git
   cd smartshift
```

2. Install dependencies
```bash
   npm install
```

3. Start development server
```bash
   npm run dev
```

4. Open http://localhost:5173 in your browser

### Build for Production
```bash
npm run build
```
